# 🪙 CryptoTrack Pro

Dashboard de criptomonedas en tiempo real construido con Next.js 16 y TypeScript.

🔗 **Live Demo:** [crypto-track-dashboard.vercel.app](https://crypto-track-dashboard.vercel.app)

## Features

- 📊 Top 100 criptomonedas en tiempo real (CoinGecko API)
- 🔄 Actualización automática cada 30 segundos
- 📈 Gráfico de precio histórico estilo TradingView (7D / 30D / 90D)
- 🔍 Búsqueda en vivo por nombre o símbolo
- ↕️ Ordenar por precio, market cap, volumen o cambio 24h
- 🌙 Diseño dark glassmorphism responsive

## Stack

- Next.js 16 + TypeScript
- Tailwind CSS
- TradingView Lightweight Charts v4
- CoinGecko API

## Run locally

\`\`\`bash
git clone https://github.com/GJomequeM/crypto-track-dashboard
cd crypto-track-dashboard
npm install
npm run dev
\`\`\`
